### 主要实现的功能

·基本的光线追踪，可调节渲染分辨率和光线反射次数

·实现基本的光照点光源与方向光

·实现基本的贴图，漫反射贴图，高光贴图，法线贴图

·多线程渲染

·通过BSP树分割场景，加速光线追踪
